# MOVARE · Gerador de Conteúdo

Gerador de posts estratégicos usando IA, baseado na Anatomia do Post Ideal da MOVARE.

## 🚀 Deploy

1. Faça fork/clone deste repositório
2. Conecte ao Vercel
3. Configure a variável de ambiente `ANTHROPIC_API_KEY`
4. Deploy!
